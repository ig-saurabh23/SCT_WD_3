# SCT_WD_3
Tic-Tac-Toe Web Application

A simple yet engaging Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

Features
- Responsive Design: Play on any device with a modern web browser.
- Single-Player Mode: Challenge the computer's AI.
- Multi-Player Mode: Compete against a friend.
- Score Tracking: Keep track of wins, losses, and draws.
- Reset Game: Start a new game at any time.

Gameplay
1. The game starts with an empty board.
2. Player X makes the first move, followed by Player O.
3. Players alternate turns.
4. A player wins when they have three in a row, column, or diagonal.
5. If all squares are filled and no player has won, the game is a draw.
